API Endpoints

  • POST /api/users/auth/login - ABHA authentication
  • GET /api/users/profile - User profile
  • GET /api/terminology/lookup - Search concepts
  • POST /api/translation/$translate - Translate concepts
  • GET /fhir/CodeSystem/namaste-ayurveda - FHIR CodeSystem
  • POST /fhir/Bundle - Create dual-coded bundle
